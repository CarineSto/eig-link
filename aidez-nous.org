#+title: Vos contributions sont les bienvenues !

Les défis EIG sont souvent l'occasion de publier des logiciels libres,
auquel chacun peut contribuer selon ses compétences et ses envies :
être bêta-testeur, aider à la traduction, corriger des bugs, etc.

Nous listons sur cette page des idées de choses que chacun peut faire
pour aider.

- Si vous voyiez une faute sur le [[https://entrepreneur-interet-general.etalab.gouv.fr/][site EIG]] n'hésitez pas à nous faire
  [[https://github.com/entrepreneur-interet-general/site-eig/issues/new][remonter le problème]] via Github.

- Vous pouvez tester l'application [[https://github.com/entrepreneur-interet-general/graph-explorer][Graph Explorer.]]

- Vous pouvez tester l'application [[https://github.com/entrepreneur-interet-general/OpenScraper][OpenScraper]].

- Vous pouvez tester [[https://archifiltre.github.io/v8/][la dernière version]] de l'application [[https://archifiltre.github.io/][Archifiltre]].

- Vous pouvez tester l'application [[https://github.com/entrepreneur-interet-general/opensignauxfaibles][OpenSignauxFaibles]].

Pour trouver tous les sites et outils publiés dans le cadre des
projets EIG, rendez-vous sur la page [[https://github.com/entrepreneur-interet-general/eig-link/blob/master/ouverture.org][« ouverture »]].
